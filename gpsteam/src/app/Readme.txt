This folder contains:
* App - a component responsible for rendering the components Header, Sidebar and Contents
* Contents - a components that contains the component <Route> which will redirect the path then renders it to a specific component
* Header - basically renders the header
* Navlist - component that contains the List of buttons that links to the path to a component that will be rendered in sideList
* SideBar - Basically contains and renders the NavList
* reducer
* routes
